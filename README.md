# Traffic Demand Prediction

## Overview
Machine Learning project for predicting traffic demand using road, weather, temporal, and geospatial features.

## Technologies
- Python
- Pandas
- NumPy
- CatBoost
- Scikit-Learn

## Feature Engineering
- Missing value handling
- Timestamp decomposition
- Peak-hour detection
- Geohash target encoding
- Cyclic time features
- Interaction features

## Model
CatBoost Regressor

## Evaluation Metric
R² Score

## Result
Leaderboard Score: 90.57
